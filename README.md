# LLama Server Wrapper

## Installation
### On macOS/Linux
`python3 -m venv .venv`

### On Windows
`python -m venv .venv`

### On macOS/Linux
`source .venv/bin/activate`

### On Windows
`.venv\Scripts\activate`

### Install the Hugginface Hub CLI for model managment
`pip install hf-cli`

## Run the Code
`./llama-server-wrapper`
